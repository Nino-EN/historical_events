# 📅 Historical Events Explorer

An interactive web app built with Python and Streamlit that allows users to explore historical events for any selected date. The app fetches real-time data from a public API and enriches results with Wikipedia information and images.

---

## 🚀 Features

- Select any **month and day** to explore historical events
- Fetches data from a public history API
- Displays event descriptions and years
- Provides external reference links (when available)
- Enhances results using Wikipedia search
- Displays related images for better context
- Interactive and user-friendly web interface

---

## 🛠️ Tech Stack

- Python
- Streamlit
- Requests
- Wikipedia API

---

## 🌐 Data Sources

- Historical events API: https://history.muffinlabs.com/
- Wikipedia (via Python library)

---

## 📸 Example Use Case

- Select: `May 12`
- View:
  - Major historical events on that date
  - Descriptions and years
  - Related Wikipedia images and links
